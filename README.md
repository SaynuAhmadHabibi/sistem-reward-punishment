# sistem-reward-punishment
Sistem Pendukung Keputusan (SPK) berbasis metode TOPSIS yang digunakan untuk membantu proses pengambilan keputusan secara objektif dan terstruktur. Sistem ini mengolah data kriteria dan alternatif, melakukan normalisasi, pembobotan, hingga menghasilkan perankingan berdasarkan nilai preferensi terbaik.

Aplikasi dikembangkan berbasis web dan dirancang untuk menampilkan alur perhitungan TOPSIS secara transparan, mulai dari input data hingga hasil akhir, sehingga memudahkan pengguna dalam melakukan evaluasi dan analisis keputusan.
